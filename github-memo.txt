git reset 커밋해시 : 커밋을 취소하는 명령어
--soft: 커밋을 롤백하고 최신내용은 남겨둠
-pull, merge, push 이런게 안되면 사용함
--hard: 커밋을 롤백하고 내용도 해당 커밋 내용으로 변경됨
- 내걸 다 날리겠다

git merge (서브)브랜치명
git branch -D 브랜치명 // 브랜치 삭제

git remote add 원격리포지토리명 원격리포지토리URL.git  //원격저장소 등록

git push 원격리포지토리이름 브랜치명 : 원격 저장소의 특정 브랜치에 저장사항 저장
git push -u " : 원격 저장소의 특정 브랜치에 변경사항 적용 (다음부터 git push 명령만으로도 해당 원격 저장소의 브랜치에 적용 가능)

git pull 원격리포지토리이름 브랜치명 : 원격 저장소의 특정 브랜치를 로컬 컴퓨터에 적용
-u 옵션 적용 가능
